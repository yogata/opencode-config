# マイグレーション構造の改善

## 1. マイグレーションディレクトリ構造

**エラー症状:**
マイグレーションが単一ファイルのみで、バージョン管理が不適切

**対処法:**
```
v2/supabase/migrations/
  001_init.sql
  002_add_staffs_table.sql
  003_add_serial_id.sql
  007_add_schedules_new_schema.sql
```

**学び:**
- マイグレーションを複数ファイルに分割する
- バージョン番号付きのマイグレーションファイルを作成する
- ファイル名は一意なプレフィックス（数値）を持たせる
